# QA瓜田项目前端仓库

## 环境准备

确保你的电脑上安装有[Node.js](https://nodejs.org/zh-cn/)

## Project setup
```
npm install cnpm -g --registry=https://registry.npmmirror.com/
cnpm install
```

### Compiles and hot-reloads for development
```
cnpm run serve
```

### Compiles and minifies for production
```
cnpm run build
```
